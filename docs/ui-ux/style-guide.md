# Руководство по стилю CommunityUpgrade

- **Темный цвет (Dark)**: #2c3e50 - Темно-синий

## Компоненты интерфейса

### Кнопки
- **Основная кнопка**: Зеленый фон (#2ecc71), белый текст, скругление 50px
- **Вторичная кнопка**: Синий фон (#3498db), белый текст, скругление 50px
- **Контурная кнопка**: Прозрачный фон, цветная граница, цветной текст, скругление 50px

### Карточки
- **Проекты**: Белый фон, легкая тень, скругление 15px
- **Профили подрядчиков**: Белый фон, легкая тень, скругление 10px

### Формы
- **Поля ввода**: Скругление 20px, легкая граница
- **Выпадающие списки**: Скругление 20px, легкая тень
- **Чекбоксы и радио-кнопки**: Кастомный стиль с основным цветом

## Иконки и иллюстрации
- **Набор иконок**: Font Awesome 6
- **Стиль иллюстраций**: Минималистичный, плоский дизайн

## Состояния элементов
- **Hover**: Легкое изменение цвета или масштаба
- **Active**: Более темный оттенок основного цвета
- **Disabled**: Сниженная непрозрачность, нет эффектов при наведении

## Анимации и переходы
- **Длительность**: 0.3s
- **Функция времени**: ease-in-out
- **Элементы с анимацией**: Кнопки, карточки, всплывающие окна

## Отзывчивый дизайн
- **Мобильный**: < 576px
- **Планшет**: 576px - 992px
- **Десктоп**: > 992px
